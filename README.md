# SMS
A nodejs SMS router using SIM800C USB module 

## Prerequisites

### Hardware
- A SIM800C USB module 
- A SIM card
- Laptop or PC 


### Software
- Needed by node-gyp to compile the serialport module:
  - Python 3
  - Visual Studio Build tools (2017)


## Install

    git clone https://github.com/kosinix/sms.git
    npm install
